SysctlValue

public protocol SysctlValue

Describes a type that can be read from / written to sysctl. Usually, no new conformances to this protocol are necessary and the set of default conformances below will suffice.

  • SysctlPointerType

    The underlyling type of the pointer read from / written to sysctl.

    Declaration

    Swift

    associatedtype SysctlPointerType
  • init(sysctlPointer:)

    Creates the type from a pointer read from sysctl.

  • withSysctlPointer(do:)

    Retrieves a pointer to write to sysctl. Note that the pointer should only be alive during the execution of work.

    Declaration

    Swift

    func withSysctlPointer<T>(do work: (UnsafePointer<SysctlPointerType>, Int) throws -> T) rethrows -> T

    Parameters

    work

    The closure during which the pointer should be alive. The second parameter should be the size of the pointer.
